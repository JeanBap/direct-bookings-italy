## Outreach Process

### Phase 1: WhatsApp (13,337 properties)
1. Set up WhatsApp Business account for AffittoDiretto.it
2. Use WhatsApp Business API (via Twilio or MessageBird) for bulk sending
3. Send in batches of 500/day to avoid spam flags
4. Personalize each message with property name and city
5. Track responses and claims
6. Estimated timeline: ~27 days for full outreach

### Phase 2: Email (15,710 properties)
1. Set up email domain (info@affittodiretto.it) with proper SPF/DKIM/DMARC
2. Use SendGrid or Mailchimp for bulk email with merge tags
3. Send in batches of 2,000/day
4. A/B test Italian vs English subject lines
5. Follow up after 7 days if no response
6. Estimated timeline: ~8 days for initial, +8 for follow-up

### Phase 3: Follow-up
1. Properties that opened email but didn't claim: send reminder after 7 days
2. Properties that claimed but didn't complete profile: send onboarding email
3. Track conversion: outreach -> claim -> first inquiry -> first booking
