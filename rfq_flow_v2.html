⚙️ State Machine
RFQ REQUEST STATES
active → Accepting quotes
matched → Quote accepted
expired → No quotes in time
cancelled → Guest cancelled
OUTREACH STATES
sent → Email delivered
quoted → Host replied with quote
declined → Host not available
expired → 24h window passed
QUOTE STATES
pending → Awaiting guest decision
accepted → Booking confirmed
declined → Guest passed
expired → Validity elapsed

Case 1: papoutsis89@gmail.com makes 1 request (Roma, Apr) → 1 host responds → guest accepts → booking confirmed ✅

Case 7: Guest accepts one quote → other pending quotes for same request automatically declined (rfq-accept edge function handles this)
Case 8: Host with 3 properties picks different one for each quote → each quote card in "My Quotes Sent" shows correct property name
Case 9: yanni@raisereadybook.com makes a request → signs in on dashboard → sees their request → timeline shows quotes arriving
Case 10: New user signs up → immediately sees empty "My Requests" and "My Quotes" with CTAs to make a request / list a property
